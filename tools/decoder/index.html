Chisel Transaction Decoder

This page reads a Ravencoin transaction as a small social object: OP_RETURN stores a direct payload, while selected burn-style outputs store indexed MacDougall / Mogwai labels as unspendable-looking addresses.

Special txid solution

The txid is the durable lookup key. A txid-named JSON file can be kept beside this page as a frozen fixture, while the same txid can also be fetched live from the Ravencoin RPC proxy. That gives the viewer two paths: local artifact for demos and tests, live ledger query for the real chain. The output addresses then act like searchable tags, not spendable recipients.
